# Omni-Strategy Master Agent

This project is an implementation of a sophisticated, multi-module trading AI.

## Directory Structure

- `1_data_gathering/`: Scripts for collecting and cleaning market data.
- `2_ai_cognitive_analysis/`: The core AI/ML models for market analysis and signal generation.
- `3_risk_and_money_management/`: Logic for calculating position sizes and managing overall portfolio risk.
- `4_execution_order_management/`: MQL5-side logic for executing and managing trades.
- `5_ai_self_evolution_audit/`: Scripts for backtesting, optimizing, and auditing the AI's performance.
- `6_communication_utilities/`: Utility scripts, including Discord notifications.
- `mql5_expert_advisor/`: The MQL5 Expert Advisor (EA) source code.
- `shared_data/`: A dedicated directory for files used in the communication between Python and MQL5.
- `main.py`: The main entry point for the Python application.
